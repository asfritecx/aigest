# Sample Project: Inventory API

This is a safe, synthetic project for testing `aigest ask` and `aigest write`.

The sample contains:

- a small C# service with inventory logic;
- an HTTP-style route adapter;
- JSON configuration with ports, feature flags, and external dependencies;
- architecture notes for documentation-summary tests;
- a small xUnit-style test file for test-discovery prompts.

There are no real secrets in this folder. Values that look like credentials are placeholders.
